Installation

If you want to use these fonts on your own device or in your own project, but aren't familar with fonts work, here's a guide on how to get it set up!

Windows

Find a font file in your file explorer, right-click, and click "Install". The font should now appear in font lists. You may need to restart some applications or your whole computer.

This guide is for windows 11. Steps may differ for other major versions of Windows.

Linux

To install for all users, copy font files into /usr/local/share/fonts. You can put them in directories within this directory for organisational purposes.

To install for just yourself, copy files into ~/.local/share/fonts.

It's generally a bad idea to install fonts to /usr/share/fonts. This directory is usually handled by your system's package manager, so you shouldn't try to change it manually.

On The Web

First, you'll need to declare a font-face. You can name it whatever you want - I chose to name it "dragon-alphabet. You can point the URL to the github repository, or (recommended) host the file on your own CDN. If you're using a file type other than .ttf, you may need to change the format from "truetype" to something else.
@font-face {
    font-family: 'dragon-alphabet';
    src: url('fonts/dragon-alphabet.ttf') format('truetype');
}

Then, you can apply your font like so:
.dragon-alphabet { font-family: 'dragon-alphabet', sans-serif; }

Dragon Alphabet

From thuum.org:

The Dragon Alphabet font has all 34 runes mapped to both the upper and lower cases. Special characters are mapped on the numbers 0-9. This is the font that is used throughout the site. It also features punctation such as periods, commas, question marks, and exclamation points.

  • dragon_alphabet.ttf

Dragon Alphabet - Numbered

From thuum.org:

The Numbered Dragon Alphabet font has all of the letters and punctuation the previous font has, in addition to runes for numbers. Special characters have been moved from the number keys to uppercase keys. Special thanks to RegalDragon for putting the number runes and font together!
