About 2024-09-07

unslaad.org is a website wrapper for the posh-crouton/dovahzul-archive GitHub repository.

On some day between the 12th and 14th of June, 2024, the website thuum.org went down. This wasn't entirely unheard of - there had been outages before - but this time, it remained down for over a month. Its maintainer, who goes by paarthurnax, can be difficult to contact, so we weren't sure if the site would ever come back online. (Thankfully, some time later, the site was restored, and remains online as of the time of writing).

In reponse to the possibility of thuum.org being lost, on the 14th of July 2024, I started this effort to preserve knowledge relating to Dovahzul. With the help of some people from the Unofficial Thuum.org Discord server, I've been collecting resources relating to the language, and building this repository with the following principles:

The archive is open-source. All archived content can be found in the repository, licenses permitting. If ever a license does not permit redistribution, I will do my best effort to catalog it in a permissible manner, and point to where it can be accessed.

The site is static. For those non-technical, this means that its hosting costs are so negligible that GitHub kindly hosts it for free on GitHub pages. It also means that there is mininal communication with a server - no personal data is collected, and each page loads once, and does not communicate with the server again.

The archive isn't going away. Or so I hope. unsl1d, undying - I have no plans to delete the content of this archive. So long as GitHub exists, and no external action is taken to take down the content here, this project will continue providing knowledge to anyone who seeks it.
